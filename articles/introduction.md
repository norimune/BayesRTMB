## BayesRTMB Package Reference Manual

### 1. Model Definition and Construction

#### `Dim(dim = 1, type = NULL, lower = NULL, upper = NULL, random = FALSE)`

Specifies the dimensions, type, bounds, and random effects of a
parameter.

- **Arguments:**
  - `dim`: Parameter dimensions (e.g., scalar=1, vector=length,
    matrix=c(rows, columns)).
  - `type`: Parameter type (e.g., `"vector"`, `"matrix"`, `"ordered"`,
    `"simplex"`, `"corr_matrix"`, `"cov_matrix"`, `"CF_corr"`). If
    omitted, it is automatically inferred from the dimensions.
  - `lower`: Lower bound of the parameter.
  - `upper`: Upper bound of the parameter.
  - `random`: If `TRUE`, treats this parameter as a random effect,
    making it subject to Laplace approximation (applicable to both MAP
    estimation and MCMC).

#### `rtmb_model(data, code, par_names = list(), init = NULL, view = NULL, null_target = NULL)`

Combines data and model definition blocks to build an R6 class model
object.

- **Arguments:**
  - `data`: Named list containing observation data and constants.
  - `code`: Model definition block generated by
    [`rtmb_code()`](https://norimune.github.io/BayesRTMB/reference/rtmb_code.md).
  - `par_names`: List of specific names corresponding to the dimensions
    of each parameter (optional).
  - `init`: List or numeric vector of initial values. If omitted, values
    are automatically generated randomly.
  - `view`: Character vector of parameter names to display
    preferentially in [`summary()`](https://rdrr.io/r/base/summary.html)
    outputs.
  - `null_target`: Used to construct a null model simultaneously;
    specify the target parameter to fix and the prior distribution to
    disable (e.g., `"delta ~ cauchy(0, r)"`).

#### `rtmb_code(...)`

The block defining the model structure. Passed to the `code` argument of
`rtmb_model`.

- **Block Types:**
  - `data`: (Optional) Block declaring variables explicitly passed from
    the `data` argument.
  - `parameters`: Block declaring model parameters using the `Dim`
    function.
  - `model`: Block defining the likelihood and prior distributions.
  - `transform`: (Optional) Block defining parameters transformed from
    the main parameters.
  - `generate`: (Optional) Block defining generated quantities (e.g.,
    predictions, log-likelihoods for WAIC) calculated from the posterior
    distribution.

### 2. Estimation Methods (Methods of `RTMB_Model`)

#### `$optimize(...)`

Performs Maximum A Posteriori (MAP) estimation or Maximum Likelihood
Estimation (MLE). \* **Arguments:** \* `num_estimate`: Number of times
to run the estimation to avoid local optima. \* `laplace`: If `TRUE`,
marginalizes random effects using Laplace approximation. \* `optimizer`:
Specifies the optimizer (`"nlminb"` or `"optim"`). \* `method`:
Optimization method when using `optim` (e.g., `"BFGS"`). \* `map`: Used
to fix parameters. Generally manipulated automatically via
`$null_model()`.

#### `$sample(...)`

Draws posterior samples using the No-U-Turn Sampler (NUTS). \*
**Arguments:** \* `sampling`: Number of sampling iterations per chain.
\* `warmup`: Number of warmup (burn-in) iterations per chain. \*
`chains`: Number of Markov chains. \* `thin`: Thinning interval. \*
`delta`: Target acceptance rate for NUTS (default: 0.8). \*
`max_treedepth`: Maximum tree depth for NUTS (default: 10). \*
`parallel`: If `TRUE`, runs chains in parallel using the `future`
package. \* `laplace`: If `TRUE`, performs NUTS sampling on the
marginalized posterior using Laplace approximation.

#### `$variational(...)`

Performs Automatic Differentiation Variational Inference (ADVI). \*
**Arguments:** \* `iter`: Number of optimization iterations. \*
`tol_rel_obj`: Convergence threshold (relative change in ELBO). \*
`num_samples`: Number of posterior samples to generate from the fitted
variational distribution. \* `num_estimate`: Number of times to run ADVI
(treated as chains to avoid local optima). \* `alpha`: Learning rate for
the optimizer. \* `laplace`: If `TRUE`, applies ADVI to the marginalized
posterior using Laplace approximation.

------------------------------------------------------------------------

### 3. Other Methods (Methods of `RTMB_Model`)

#### `$null_model(target, value = 0)`

Creates a new model object with the specified parameter fixed to `value`
(useful for calculating Bayes Factors). `target` can be specified as a
string like `"delta"` or `"delta ~ cauchy(0, 1)"`.

#### `$print_code()`

Prints the internal model structure (parameter list and `log_prob`
function code) to the console.

------------------------------------------------------------------------

### 4. Fit Result Objects

#### `MAP_Fit` Class

Stores the results of `$optimize()`. \* **Fields:** `par` (parameter
list on the constrained scale), `objective` (negative log-posterior),
`log_ml` (log marginal likelihood via Laplace approximation), `df_fixed`
/ `random_effects` (summary data frames of estimates). \* **Methods:**
\* `$summary(max_rows = 20, digits = 2)`: Outputs a table summarizing
estimates, standard errors, and 95% Wald confidence intervals.

#### `MCMC_Fit` Class

Stores the results of `$sample()` and performs post-processing of
posterior samples. \* **Methods:** \* `$summary(pars = NULL, ...)`:
Returns a data frame of summary statistics including Rhat, effective
sample size (ESS), posterior mean, and credible intervals. \*
`$draws(pars = NULL, chains = NULL, inc_random = FALSE, inc_transform = TRUE, inc_generate = TRUE)`:
Extracts posterior samples as a 3D array `(iteration, chain, variable)`.
\* `$bridgesampling(method = "warp3", seed = NULL)`: Estimates the
marginal likelihood via Bridge Sampling. \*
`$generated_quantities(code)`: Computes generated quantities (GQ) for
the extracted posterior samples and stores them in the object.

#### `VB_Fit` Class

Stores the results of `$variational()`. \* **Methods:** \*
`$summary(pars = NULL, ...)`: Returns a data frame of summary statistics
(Note: Rhat and ESS are not computed). \*
`$draws(pars = NULL, best_only = TRUE, ...)`: Extracts posterior
samples. If `best_only = TRUE`, extracts samples only from the run with
the highest ELBO. \* `$plot_elbo(tail_n = 1000)`: Plots the Evidence
Lower Bound (ELBO) history during optimization. \*
`$plot_trajectory(pars = NULL)`: Plots the parameter trajectory in the
final optimization window. \* `$generated_quantities(code)`: Computes
generated quantities.
