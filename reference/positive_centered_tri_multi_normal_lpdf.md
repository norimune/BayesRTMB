# Positive centered triangular multivariate normal log-probability density function

Positive centered triangular multivariate normal log-probability density
function

## Usage

``` r
positive_centered_tri_multi_normal_lpdf(x, sigma = 1)
```

## Arguments

- x:

  Matrix of quantiles (centered triangular matrix with positive
  diagonals).

- sigma:

  Standard deviation parameter(s).

## Value

The log-density.
