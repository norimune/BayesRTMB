# Sufficient statistics multivariate normal log-probability density function (covariance parameterization)

Sufficient statistics multivariate normal log-probability density
function (covariance parameterization)

## Usage

``` r
sufficient_multi_normal_lpdf(S_mat, N, y_bar, mean, Sigma)
```

## Arguments

- S_mat:

  Deviation sum of squares matrix.

- N:

  Sample size.

- y_bar:

  Sample mean vector.

- mean:

  Mean parameter vector.

- Sigma:

  Covariance matrix.

## Value

The exact log-likelihood of the N raw observations.
