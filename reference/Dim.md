# Define parameter dimensions and types

Define parameter dimensions and types

## Usage

``` r
Dim(dim = 1, type = NULL, lower = NULL, upper = NULL, random = FALSE)
```

## Arguments

- dim:

  Dimensions of the parameter.

- type:

  Type of the parameter.

- lower:

  Lower bound.

- upper:

  Upper bound.

- random:

  Logical; whether it is a random effect.
