# LKJ correlation log-probability density function for Cholesky factors

LKJ correlation log-probability density function for Cholesky factors

## Usage

``` r
lkj_CF_corr_lpdf(CF_Omega, eta = 1)
```

## Arguments

- CF_Omega:

  Cholesky factor of a correlation matrix.

- eta:

  Shape parameter.

## Value

The log-density.
