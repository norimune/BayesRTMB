# Best-Worst Categorical Logit Log-PMF (RTMB optimized)

Best-Worst Categorical Logit Log-PMF (RTMB optimized)

## Usage

``` r
bw_categorical_logit_lpmf(x, U, lambda = 1)
```

## Arguments

- x:

  Vector of indices of the selected pairs.

- U:

  Vector of utilities for each item.

- lambda:

  Scaling parameter.

## Value

Log-probability (scalar advector).
