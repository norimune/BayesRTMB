# Log mixture of two probabilities

Log mixture of two probabilities

## Usage

``` r
log_mix(theta, lp1, lp2)
```

## Arguments

- theta:

  Mixing proportion (between 0 and 1).

- lp1:

  Log-probability of the first component.

- lp2:

  Log-probability of the second component.

## Value

The log-probability of the mixture.
