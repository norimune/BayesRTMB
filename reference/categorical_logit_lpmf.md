# Categorical log-probability mass function with logit parameterization

Categorical log-probability mass function with logit parameterization

## Usage

``` r
categorical_logit_lpmf(x, eta)
```

## Arguments

- x:

  Integer or integer vector. The observed category index.

- eta:

  A numeric vector of linear predictors (unnormalized
  log-probabilities).

## Value

The sum of the log-probability.
