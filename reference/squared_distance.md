# Squared Euclidean distance

Squared Euclidean distance

## Usage

``` r
squared_distance(x, y, eps = 1e-08)
```

## Arguments

- x:

  A numeric vector.

- y:

  A numeric vector.

- eps:

  A small value added for numerical stability (default is 1e-8).

## Value

The squared Euclidean distance between x and y.
