# 因子負荷量をソートして見やすく表示する関数

因子負荷量をソートして見やすく表示する関数

## Usage

``` r
sort_loadings(loadings, cutoff = 0, round_digits = 3)
```

## Arguments

- loadings:

  因子負荷量の行列、データフレーム、またはそれらを含むリスト（リストの場合は最初の要素を使用）

- cutoff:

  負荷量の絶対値がこの値未満の場合は表示を空白にする（デフォルトは0.3）

- round_digits:

  小数点以下の表示桁数（デフォルトは3）

## Value

並び替えられた負荷量行列（不可視で返すため、変数への代入も可能）
