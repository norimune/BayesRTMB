# Logit-Normal log-probability density function

Logit-Normal log-probability density function

## Usage

``` r
logit_normal_lpdf(x, mu, sd)
```

## Arguments

- x:

  Vector of quantiles (must be strictly between 0 and 1).

- mu:

  Vector of means on the logit scale.

- sd:

  Vector of standard deviations on the logit scale.

## Value

The sum of the log-density.
