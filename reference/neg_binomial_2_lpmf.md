# Negative binomial log-probability mass function (alternative parameterization)

Negative binomial log-probability mass function (alternative
parameterization)

## Usage

``` r
neg_binomial_2_lpmf(x, mu, size)
```

## Arguments

- x:

  Vector of quantiles.

- mu:

  Mean parameter.

- size:

  Dispersion parameter.

## Value

The sum of the log-probability.
