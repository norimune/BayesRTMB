# Multivariate Cauchy log-probability density function

Multivariate Cauchy log-probability density function

## Usage

``` r
multi_cauchy_lpdf(x, mean, Sigma, sum = TRUE)
```

## Arguments

- x:

  Vector or matrix of quantiles.

- mean:

  Vector or matrix of means.

- Sigma:

  Scale matrix.

## Value

The sum of the log-density.
