# Binomial log-probability mass function with logit parameterization

Binomial log-probability mass function with logit parameterization

## Usage

``` r
binomial_logit_lpmf(x, size, eta)
```

## Arguments

- x:

  Vector of quantiles (number of successes).

- size:

  Number of trials.

- eta:

  Linear predictor (logit probability).

## Value

The sum of the log-probability.
