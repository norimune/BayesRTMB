# Multivariate normal log-probability density function

Multivariate normal log-probability density function

## Usage

``` r
multi_normal_lpdf(x, mean, Sigma)
```

## Arguments

- x:

  Vector or matrix of quantiles.

- mean:

  Vector or matrix of means.

- Sigma:

  Covariance matrix.

## Value

The sum of the log-density.
