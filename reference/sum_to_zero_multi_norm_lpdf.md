# Sum-to-zero multivariate normal log-probability density function

Sum-to-zero multivariate normal log-probability density function

## Usage

``` r
sum_to_zero_multi_norm_lpdf(x, sigma = 1, K = NULL)
```

## Arguments

- x:

  Vector of quantiles.

- sigma:

  Standard deviation parameter.

- K:

  Dimension. If NULL, inferred from the length of x.

## Value

The log-density.
