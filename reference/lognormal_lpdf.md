# Lognormal log-probability density function

Lognormal log-probability density function

## Usage

``` r
lognormal_lpdf(x, meanlog, sdlog)
```

## Arguments

- x:

  Vector of quantiles.

- meanlog:

  Mean of the distribution on the log scale.

- sdlog:

  Standard deviation of the distribution on the log scale.

## Value

The sum of the log-density.
