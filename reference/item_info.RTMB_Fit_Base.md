# RTMB_Fit_Base の項目情報関数

RTMB_Fit_Base の項目情報関数

## Usage

``` r
# S3 method for class 'RTMB_Fit_Base'
item_info(x, theta_seq = seq(-4, 4, length.out = 100), items = NULL, ...)
```

## Arguments

- x:

  An object of class RTMB_Fit_Base.

- theta_seq:

  評価する特性値（能力値）のシーケンス。

- items:

  計算対象を特定の項目に絞る場合のインデックスまたは項目名（省略可能）。

- ...:

  Additional arguments.
