# Multivariate Student-t log-probability density function

Multivariate Student-t log-probability density function

## Usage

``` r
multi_student_t_lpdf(x, df, mean, Sigma, sum = TRUE)
```

## Arguments

- x:

  Vector or matrix of quantiles.

- df:

  Degrees of freedom.

- mean:

  Vector or matrix of means.

- Sigma:

  Scale matrix.

## Value

The sum of the log-density.
