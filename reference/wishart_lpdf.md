# Wishart log-probability density function

Wishart log-probability density function

## Usage

``` r
wishart_lpdf(X, n, V)
```

## Arguments

- X:

  Symmetric positive-definite matrix (scatter matrix).

- n:

  Degrees of freedom (sample size N).

- V:

  Scale matrix (covariance matrix Omega).

## Value

The log-density with all constant terms.
