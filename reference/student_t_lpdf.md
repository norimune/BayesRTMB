# Student-t log-probability density function

Student-t log-probability density function

## Usage

``` r
student_t_lpdf(x, df, mu = 0, sigma = 1)
```

## Arguments

- x:

  Vector of quantiles.

- df:

  Degrees of freedom.

- mu:

  Location parameter.

- sigma:

  Scale parameter.

## Value

The sum of the log-density.
