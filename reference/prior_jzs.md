# Specify a JZS (Jeffrey-Zellner-Siow) prior for t-tests

Specify a JZS (Jeffrey-Zellner-Siow) prior for t-tests

## Usage

``` r
prior_jzs(r = 0.707)
```

## Arguments

- r:

  Numeric; scale parameter for the Cauchy prior on effect size delta.
  Default is 0.707.

## Value

A list with class "rtmb_prior"
